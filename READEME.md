# 📚 托业背单词 - TOEIC Vocabulary Builder

一款基于 Web 的托业（TOEIC）词汇学习工具，内置 **540 个高频托业词汇**，采用**间隔重复（Spaced Repetition）**算法，帮助你高效记忆单词。

> 纯前端项目，无需后端服务，数据存储在浏览器 `localStorage` 中，打开即用。

---

## 🚀 快速开始

### 方式一：直接打开（推荐）

1. 将整个项目文件夹下载到本地
2. 双击打开 `index.html`（主入口）
3. 即可开始使用，无需安装任何依赖



---

## 📖 使用说明

### 1. 主页仪表盘（`index.html`）

打开主页后，你可以看到：

- **今日任务**：显示待复习单词数、新单词数和已完成数，以及今日完成度进度条
- **学习进度**：显示总单词数、已掌握数和整体掌握率
- **三个导航入口**：
  - 📖 **开始背诵** → 进入单词学习页面
  - 📝 **卡片管理** → 管理你的单词卡片
  - 📈 **进度统计** → 查看学习数据分析

### 2. 开始背诵（`study.html`）

核心学习页面，以**翻转卡片**的方式背诵单词：

- 点击卡片或按 **空格键 / Enter** 翻转查看释义
- 掌握程度评分：按 **1**（忘记）、**2**（困难）、**3**（良好）
- 按 **← →** 方向键切换上/下一个单词
- 按 **S** 跳过当前卡片
- 页面右侧提供快捷搜索按钮，支持 **Bing 图片搜索**、**有道词典**、**豆包 AI** 等多渠道辅助查询

### 3. 背诵模式（`memorization.html`）

Anki 风格的背诵模式，采用翻转卡片 + 评分机制：

- 点击卡片翻转查看释义
- 根据掌握情况选择：**Again（再来）/ Hard（困难）/ Good（良好）/ Easy（简单）**
- 系统会根据你的评分自动调整下次复习时间

### 4. 卡片管理（`cards.html`）

管理你的单词卡片库：

- **搜索**：按单词或释义搜索
- **筛选**：按掌握状态（新词/学习中/已掌握）或标签筛选
- **编辑**：点击卡片可编辑正面（单词）和背面（释义）
- **删除**：移除不需要的卡片
- **添加**：手动添加自定义单词卡片

### 5. 进度统计（`statistics.html`）

可视化展示你的学习数据：

- 总单词数、已掌握数、学习中、未开始等统计卡片
- 按时间范围筛选查看学习趋势
- 按标签维度分析各课程的学习进度

### 6. 词汇手册（`toeicIndex.html`）

完整的托业词汇参考手册：

- 按课程分类浏览所有 540 个托业词汇
- 支持侧边栏课程导航快速切换
- 支持全局搜索单词
- 每个单词包含：音标、词性、中文释义、例句、同义词、派生词、常见搭配

### 7. 重置数据（`reset-cards.html`）

如果需要重新开始：

- 打开 `reset-cards.html`
- 点击「重置并导入托业词汇」按钮
- 将清除所有学习进度并重新加载完整的 540 个词汇
